Purpose of this Page

A site map is planning tool for web design that lists the pages on a web site, organizing them in hierarchical fashion.  They are used to show the relationship between the pages, display user pathways, and if possible, indicate which templates used for each page.  Throughout the process they are constantly updated to reflect any changes made to the site’s navigation structure or classification scheme.
